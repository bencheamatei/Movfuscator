# Movfuscator-Proiect-ASC

Benchea Matei - Grupa 152
Mitoceanu Ciprian - Grupa 152

Trebuie scris un program care primeste ca input un program assembly x86 si il transforma intr-un program echivalent care sa foloseasca cat de mult se poate instructiunea mov. Initial vreau sa inlocuiesc doar operatiile aritmetice si logice. Adding to this, vreau sa pot sa fac operatii pe 32 de biti, nu doar pe 8 cu lookup tables (urmeaza a fi vazut daca e un goal realistic).
